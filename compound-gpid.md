# povineq

## Objective

Python PIP API wrapper. It allows users to download data from the PIP API in the same way that the `pipr` R package does.

## Key Deliverables

- A functional and publishable Python package that anybody can use to interact with the PIP API

## Constraints

- Must follow all the features that the current `pipr` R package has
- Must be very efficient
